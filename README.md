# 📝 Priority Based To-do List

## 📌 Project Description

This is a Python-based Task Manager application that allows users to manage their daily tasks efficiently. It is a console-based application built using core Python concepts.

The project demonstrates operations like (Create, Read, Update, Delete) along with file handling for data persistence.

---

## 🚀 Features

* ✅ Add new tasks (Create)
* 📋 View all tasks (Read)
* ✏️ Edit existing tasks (Update)
* ❌ Delete tasks (Delete)
* ✔️ Mark tasks as completed
* 🔍 Search tasks
* ⚡ Priority system (High / Medium / Low)
* 🗑️ Delete all tasks
* 📊 Task statistics (Total / Pending / Completed)
* 🕒 Timestamp for each task
* 💾 File handling (data stored in `tasks.txt`)
* ⚠️ Exception handling for invalid inputs

```

↓
```
